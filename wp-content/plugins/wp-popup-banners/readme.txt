<strong>WP Popup Banners</strong> is a FREE Popup plugin for WordPress. You can use this to add popups to your site with custom images or messages with different effects.

* <strong>Create your image</strong> which you want to display on popup.
* <strong>Define your content </strong>– configure which content you’d like to show as popup

### Features ###
- <strong>Unlimited Popup</strong>
- <strong>Text and Image Popup</strong>
- <strong>WYSIWYG Editor for Text Popup to configure the popup components as per your need</strong>
- <strong>General Settings</strong>
    - Popup Animation
    - Show Every Time or Once
    - Disable on mobile feature
    - Enable Popup Delay - Show popup after some time
    - Enable Auto Close - Auto close after certain time
    - Page wise filter for popup display
- <strong>Display Settings</strong>
    - Width of Popup
    - Height or Popup
    - Popup Display Type - Fixed or Absolute
    - Shape - Rounded or Sharp Corner
    - Overlay Configuration Options
    - Close Button - Enable or Disable
    - Background Configuration Options
    - Border Configuration Options
- Preview Options Available
- <strong>User Friendly Interface</strong>
- <strong>Mobile Friendly</strong>
- <strong>Translation Ready</strong>

== Installation ==
1. Unzip wp-popup-banners.zip
1. Upload all the files to the /wp-content/plugins/wp-popup-banners
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. For creating new popup, click on the Add New button on Manage Popups menu.


== Frequently Asked Questions ==

= What does this plugin do? =
This plugin provides the ability to popup your custom images or contents on page load. 
We can display the popup on the pages according to our requirement. For example we can 
display popup only on home page or all pages or only the specific pages. We can confugure 
popup delay and autoclose with message.
